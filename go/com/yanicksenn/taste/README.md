# Taste

Taste is a small tool to generate programming language agnostic data structures.

NOTE: This is work in progress and does not fully work yet :)

## Build

```
go build main.go
```

## Run 

```
./main sample.taste
```

```
# sample.taste

namespace taste.lang;

type MyType {
    string name;
    int counter;
    bool is_active;
}
```
